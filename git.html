git에 파일 옮기기

1. 내 파일을 git과 연동시키키 -> git config --global user.name mk0537
, git config --global user.email mk0537@naver.com
2. git 시작 -> git init
3. 로컬컴퓨터 (내PC)에 있는 파일을 스테이지(stage)로 옮기기 -> git add .
4. 스테이지(stage)에서 레파지토리(repo)로 옮기기 -> git commit -m "first commit"
5. 형상관리(버전관리) 업로드 -> git remote add origin https://github.com/mk0537/instagram.git
